# Analyse-du-systeme-educatif-ressources-genre-et-performance-2010-2025-
Ce projet analyse l’évolution du système éducatif sur la  période de 2010 à 2025 en mettant l’accent sur les ressources éducatives, la scolarisation des filles et la performance scolaire, à travers une approche descriptive basée sur des outils de data visualisation.
